# A&J Personal Collections ✦ Curated Lifestyle Discovery

*Simple, practical design and everyday essentials curated for style, home, and daily life.*

Welcome to the home of A&J Personal Collections — a curated product discovery platform run as a joint team by Anna and John. What began as a simple, long-distance exchange of care packages between Australia and the Philippines has grown into a shared space where we hand-select and recommend clothing, homewares, and daily essentials that we use in our own household.

---

## 🌐 Live Site


- **Hosting:** [Netlify](https://www.netlify.com) (connected via continuous deployment)
- **Domain:** [Porkbun](https://www.porkbun.com)
- **Philosophy:** Built on vanilla web standards — semantic HTML5, clean CSS Grid/Flexbox, and zero-dependency JavaScript.

---

## ✨ Core Features

- **Dynamic Product Catalog:** Loads and filters curated lifestyle items (apparel, homewares, and beauty) on the fly directly from a local database (`json/products.json`).
- **Responsive Category Filtering:** Offers a clean vertical navigation sidebar on desktop screens and a horizontal swipe-bar on mobile viewports.
- **Aesthetic Grid Collage Hero:** Responsive flex-grid collage on the home page utilizing your high-quality local WebP assets.
- **Interactive Modals:** 
  - **Detail Modal:** Displays prices, descriptions, promo codes, and a direct link to the external merchant.
  - **Purchase Guidelines Popout:** Accessible, mobile-optimized popup modal pinned to the header containing transaction instructions and your SHEIN app search code (`ZVE2Y`).
- **Pure Discovery Curation:** A fully transparent model routing visitors safely to third-party merchant platforms at zero extra cost.

---

## 🧰 Built With

- **HTML5 & CSS3** — Semantic elements, responsive CSS Grid, Flexbox alignment, mobile layouts, and custom scrollbar aesthetics.
- **Vanilla JavaScript** — Fast, client-side data binding, category sorting, and modal DOM manipulation with zero framework overhead.
- **JSON Database** — Easily maintainable and offline-ready local catalog storage (`products.json`).
